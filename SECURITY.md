## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1. **Do NOT create a public issue.**
2. Email us at **<security@bucchain.com>** with details of the vulnerability.
3. Include steps to reproduce the issue.
4. We will acknowledge your report within 48 hours and provide a timeline for a fix.

Thank you for responsibly disclosing security issues!
